# My first post

So, I just finished working on this very website. I went through a lot to be able to build a project like this from scratch in C.

First, I created a single-header library, [htmpl.h](https://github.com/me7alix/htmpl.h), which compiles HTML templates into C. To render posts, which are simply Markdown documents, I also created another single-header library, [mdp.h](https://github.com/me7alix/mdp.h), which parses Markdown.
